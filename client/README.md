# Getting Started
## 0. Need Environment
- <a href="https://nodejs.org/en">Node.js 18.17</a> or later.
- <img src="https://img.shields.io/npm/v/npm.svg?logo=npm">

<a src=https://nodejs.org/en/download/prebuilt-installer>Click to Install Node.js</a>

## 1. Install
```bash
cd client
npm install
```

## 2. Running app
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
